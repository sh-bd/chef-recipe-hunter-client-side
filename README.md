## Run Locally

Clone the project

```bash
  git clone https://github.com/programming-hero-web-course-4/b7a10-chef-recipe-hunter-client-side-sh-bd.git
```

Go to the project directory

```bash
  cd b7a10-chef-recipe-hunter-client-side-sh-bd
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
