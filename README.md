## Run Locally

Clone the project

```bash
  git clone https://github.com/sh-bd/chef-recipe-hunter-client-side.git
```

Go to the project directory

```bash
  cd chef-recipe-hunter-client-side
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
